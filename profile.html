Account profile hub

One account. Two profile views.

Every account has a patron profile. Accounts with creator access also get a My Expert Profile workspace for the public bio, forum name, tier, and expert-facing identity.

Expert tools enabled This mock account is a Founding Expert, so both profile views are available. If the account is not a creator, the My Expert Profile button is disabled.
The disabled button shows the non-expert account state. In production, users would only see the enabled expert button if their account has Contributor, Verified Creator, or Founding Expert status.
Recommended tier naming: Contributor -> Verified Creator -> Founding Expert. This avoids calling the middle tier "Expert" when the whole platform already describes all creators as experts.
